## 🔍 Troubleshooting Common Issues

Check these items if you encounter problems during the process.

**The installer does not open**
Verify your internet connection and try the download again. A corrupted file prevents the installer from starting.

**The DAW does not see the plugins**
Open your DAW settings. Locate the folder path for VST plugins. Ensure the path matches the location you chose during installation. Some programs require a rescan of the plugin folder to recognize new files.

**Audio sound is distorted**
Audio plugins use processor power. If your CPU usage is too high, the audio might stutter or distort. Increase the buffer size in the audio settings of your DAW to resolve this issue.

**The window looks too small**
Right-click the plugin window. Select the scaling option to make the interface larger. This feature helps if you use a high-resolution screen.

**Plugin is missing presets**
The installer includes a bank of presets for every plugin. If you do not see them, reinstall the bundle and ensure you check the box for factory content during the installation phase.

**System freezes during installation**
Close all background applications before running the installer. High memory usage from other programs stops the installer from writing files to your hard drive.

**Does it impact computer performance?**
These plugins use average CPU resources. You can run several instances on modern machines without issues. Track the performance meter in your DAW to monitor resource usage.

**What is the difference between mono and stereo plugins?**
Some plugins come in mono and stereo versions. Use the mono version for individual instrument tracks like vocals or bass. Use the stereo version for buses or master tracks to maintain the stereo field.
